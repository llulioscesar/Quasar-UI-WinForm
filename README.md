# UI

Esta libreria esta inspirada en la interfaz de IOS para React

## Instalacion
**NPM**
~~~
npm install @jcesar.co/ui
~~~
**YARN**
~~~
yarn add @jcesar.co/ui
~~~

## implementacion
```JSX
...
import {Button} from '@jcesar.co/ui'
...
render() {
    ...
    <Button text='Click' onClick={() => console.log('click button')}/>
    ...
}
```
